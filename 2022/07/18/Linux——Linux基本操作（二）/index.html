Linux——Linux基本操作（二）

Linux——Linux基本操作（二）

vim编辑器

Linux系统都会自带vi编辑器，但是vi编辑器太难用了！所以建议大家安装vim编辑器，安装命令：sudo apt-get install vim

vi xxx 使用vi编辑器打开文件。

vim编辑器三种工作模式

  • 一般模式(指令模式)：默认模式，用vi打开一个软件以后自动进入到此模式。

  • 编辑模式：一般模式中无法编辑文件，要编辑文件就要进入编辑模式，按下“i、I、a、A、o、O、s、r”等就会进入到编辑模式。一般按下“a”进入编辑模式。按下ESC键可退出编辑模式。

  • 命令行模式(底行模式)：先进入到一般模式，然后输入：、/、？这三个中的任意一个就可以进入到命令行模式。输入/xxx，表示在文件中查找xxx；输入:wq来保存退出。:q退出，:q!不保存退出，:w保存。

其他操作方式

按下键盘上的上下左右键来控制光标。

一般模式下：

  • dd，删除光标所在行，ndd，删除光标所在行及其下n行。
  • u，撤销，恢复上一步。
  • . ,重复前一个操作。
  • yy，复制光标所在行
  • nyy，复制光标所在向下n行，比如10yy就是复制光标下10行。
  • p和P，p为复制到光标下一行，P复制到光标上一行。

编写C程序

使用VIM编辑器编写程序，也可以使用vscode。

设置vim编辑器

设置vim编辑器，一个TAB=4个字节。 设置vim编译器，显示行号。 sudo vi /etc/vim/vimrc //打开配置文件

文件末尾添加：

set ts=4
set nu

编译C程序

使用gcc编译器编译C程序。

gcc main.c //生成a.out可执行文件
gcc -c main.c //生成main.o目标文件
gcc main.c -o main //生成main可执行文件

make工具和Makefile文件

make：一般说GNU Make，是一个软件，用于将源代码文件编译为可执行的二进制文件，make工具主要用于完成自动化编译。make工具编译的时候需要Makefile文件提供编译文件。

Makefile：make工具所使用的文件，Makefile指明了编译规则。

Makefile语法

  • 规则格式

    目标  ... ：依赖 ...
	命令
	命令
	... ...

    命令列表中的每条命令必须以TAB键开始，不能使用空格！

  • 变量的使用

    • 变量的定义与引用：
    objects = main.o input.o calcu.o
main: $(objects)
	gcc -o main $(objects)
    • 赋值符“=”：相当于C语言字符串的浅copy
    name = zzk
curname = $(name)
name = zuozhongkai		//name修改，则curname跟着变化
print:
	@echo curname: $(curname)	//echo打印变量信息，@不打印执行过程

    • 赋值符“:=”：相当于C语言字符串的深copy

    • 赋值符“?=”：如果变量前面没有被赋值就赋该值，否则就使用前面赋的值

    • 变量追加“+=”：相当与python中的字符串相加

  • 模式规则：在规则的目标定义中包涵 “%”

    %.o : %.c
	#命令

  • 自动化变量

    自动化变量 描述
    $@ 规则中的目标集合，在模式规则中，如果有多个目标的话，“$@”表示匹配模式中定义的目标集合。
    $< 依赖文件集合中的第一个文件，如果依赖文件是以模式(即“%”)定义的，那么“$<”就是符合模式的一系列的文件集合。
    $^ 所有依赖文件的集合，使用空格分开，如果在依赖文件中有多个重复的文件，“$^”会去除重复的依赖文件，值保留一份。 
    objects = main.o input.o calcu.o
main: $(objects)
	gcc -o main $(objects)
	
%.o : %.c
	gcc -c $<

  • 伪目标：主要是为了避免 Makefile中定义的执行命令的目标和工作目录下的实际文件出 现名字冲突。

    .PHONY : clean    //若路径下存在clean文件，执行make clean时依然会执行下面的命令
clean:
	rm *.o
	rm main

  • 条件判断

    <条件关键字 >
	<条件为真时执行的语句 >
endif
    <条件关键字 >
	<条件为真时执行的语句 >
else
	<条件为假时执行的语句 >
endif

    其中条件关键字有 4个：ifeq、 ifneq、 ifdef和 ifndef。

    ifeq用来判断是否相等， ifneq就是判断是否不相等， ifeq用法如下：

    ifeq (<参数 1>, <参数 2>)	//括号可以用引号来表示

    ifdef用来判断是否非空， ifndef就是判断是否为空， ifdef用法如下：

    ifdef <变量名 >

  • 函数使用：Makefile中的函数只能包括特定的几个，无法编写自定义函数，具体函数使用可查询百度。

shell脚本

shell脚本类似windows的批处理文件，shell脚本就是将连续执行的命令写成一个文件。shell脚本提供数组、循环、条件判断的等功能。

脚本写法

shell脚本是个纯文本文件，扩展名为.sh，第一行一定要为:

#!/bin/bash		
# 表示使用bash。

脚本语法

  • 交互式shell脚本（read命令）

    #!/bin/bash
# -p打印提示信息
read -p "input your age and height:" age height
# $取变量的值
echo "your age=$age,your height=$height“

  • shell脚本的数值计算：shell仅支持整形，数值计算使用$((表达式))。

    #!/bin/bash
echo "Please input two intergers"
read -p "first interger:" firstInt
read -p "second interger:" secondInt
# =号两边不能存在空格
total=$(($firstInt + $secondInt))
echo "first interger:$firstInt,second interger:$secondInt“
echo "total:$total"

  • test命令：test命令用于查看文件是否存在、权限等信息，可以进行数值，字符，文件三方面的测试。

    #!/bin/bash
echo "Please input file name:"
read filename
# -e表示判断filename是否存在
test -e $filename && echo "$filename exist" || echo "$filename no exist"

  • &&和||命令：

    • cmd1 && cmd2 当cmd1执行完并且正确，那么cmd2开始执行，如果cmd1执行完毕错误，那么cmd2不执行。

    • cmd1 || cmd2 当cmd1执行完毕并正确，那么cmd2不执行，反之cmd2执行。

  • 中括号[]判断符：有test命令相似的作用，但只能判断 == 或 != 。

    #!/bin/bash
echo "Please input two params:"
read -p "first param:" firstParam
read -p "second param:" secondParam
# [空格  空格]，变量的引用需要用双引号引起来
# 变量引用的作用类似与宏定义
[ "$firstParam" == "$secondParam" ] && echo "$firstParam == $secondParam" || echo "$firstParam != $secondParam"

  • 默认变量

    ./shellScript.sh a b

    \(0~\)n，表示shell脚本的参数，包括shell脚本命令本身，shlle脚本命令本身为$0

    $#：#表示最后一个参数的标号。

    $@：表$1、$2、$3......

    #!/bin/bash
# 第一个参数即文件名
echo "file name:" $0
# 参数的数量（不包括文件名）
echo "total param num:" $#
# 所有参数（不包括文件名）
echo "whole param:" $@
echo "first param:" $1
echo "second param:" $2

脚本条件判断

  • if then elif else条件判断语句，写法：

    #!/bin/bash
read -p "Please input [Y/N]:" value
if [ "$value" == "Y" ] || [ "$value" == "y" ]; then
	echo "your input is Y"
elif [ "$value" == "N" ] || [ "$value" == "n" ]; then
	echo "your input is N"
else
	echo "your input can't identify!!"
fi

  • case语句，写法：

    #!/bin/bash
case $1 in
	“a”)
        echo "first param:a"
		;;
	“b”)
		echo "first param:b"
		;;
	*)
		echo "your input can't identify!!"
		;;
esac

循环

  • while do done：当条件成立的时候就一直循环，直到条件不成立。

    #!/bin/bash
# 类似于C语言的do while，先执行一次循环体再判断
while [ ”$value“ == "close" ]
do
	read -p "Please input a str:" value
done

echo "Stop while!!"

  • until do done：当条件不成立的时候循环，条件成立以后就不循环

  • for var in con1 con2 con3…conN：var从con1开始到conN

    #!/bin/bash
for var in con1 con2 con3
do
	echo "var is:$var"
done

  • for循环数值处理

    #!/bin/bash
read -p "Please input count:" count
total=0
# for((初始值; 限制值; 执行步长))
for((i=0; i<=count; i=i+1))
do
	total=$(($total + $i))
done
echo "1+2+...+$count=$total"

脚本函数

  • shell脚本也支持函数，函数写法如下：

    #!/bin/bash
function print() {
	echo "first param:$1,second param:$2"
}
# 函数调用，直接写函数名，a和b是传入的两个参数
print a b
